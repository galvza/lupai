**Task A: `discoverFromGoogleSearch`**
- `id: 'discover-google-search'`, `maxDuration: 120`
- Import `scrapeGoogleSearch` from `'@/lib/apify/google-search'`
- Build 3 search queries using payload: `['${niche} ${region}', '${segment} ${region} empresa', '${niche} ${region} loja site']`
- Call `scrapeGoogleSearch(queries)`
- Map results to `RawCompetitorCandidate[]` with `source: 'google-search'`
- Return the array
- Wrap in try/catch, on error log message and return empty array (independent failure per D-23)

**Task B: `discoverFromGoogleMaps`**
- `id: 'discover-google-maps'`, `maxDuration: 120`
- Import `scrapeGoogleMaps` from `'@/lib/apify/google-maps'`
- Call `scrapeGoogleMaps(niche, region)` — this returns `GmbData | null`
- The Google Maps actor returns an array of places. HOWEVER, the existing `scrapeGoogleMaps` wrapper returns only the first item as `GmbData`. For discovery, we need multiple results. So this task must call the Apify actor DIRECTLY (like in google-search.ts) with `searchStringsArray: [niche + ' ' + region]` and extract multiple results.
- Create a new ApifyClient, call `compass/google-maps-scraper`, get items, map each to `RawCompetitorCandidate`:
  ```typescript
  { name: item.title ?? '', url: item.website ?? '', description: (item.categories ?? []).join(', '), source: 'google-maps' as const }
  ```
- Filter out entries where `url` is empty (GMB listings without websites)
- Return `RawCompetitorCandidate[]`
- Wrap in try/catch, on error return empty array

**Task C: `discoverFromFacebookAds`**
- `id: 'discover-facebook-ads'`, `maxDuration: 120`
- The existing `scrapeFacebookAds` wrapper returns `MetaAdsData` (ads for a single company). For discovery, we need to find WHICH companies are advertising in this niche.
- Call the Apify actor directly: `apify/facebook-ads-scraper` with `{ searchQuery: niche + ' ' + region, country: 'BR' }`
- Map items to `RawCompetitorCandidate[]`:
  ```typescript
  { name: item.pageName ?? item.advertiserName ?? '', url: item.pageUrl ?? item.advertiserUrl ?? '', description: (item.adText ?? item.bodyText ?? '').slice(0, 200), source: 'facebook-ads' as const }
  ```
- Filter out entries where both `name` and `url` are empty
- Deduplicate by advertiser name (same advertiser may have multiple ads)
- Return `RawCompetitorCandidate[]`
- Wrap in try/catch, on error return empty array

**Task D: `discoverFromSimilarWeb`**
- `id: 'discover-similarweb'`, `maxDuration: 120`
- SimilarWeb is a validation/enrichment source, not a primary discovery source (per Research Open Question 1)
- This task receives an optional `seedUrl` in the payload (extend `DiscoveryPayload` with `seedUrl?: string`)
- If no seedUrl provided, return empty array immediately
- If seedUrl provided, call the Apify actor directly: `tri_angle/similarweb-scraper` with `{ urls: [seedUrl] }`
- Look for `similarSites` field in the response; if present, map to `RawCompetitorCandidate[]`
- If `similarSites` field not present, return empty array (graceful degradation)
- Wrap in try/catch, on error return empty array

All 4 tasks:
- Must have JSDoc with `@param payload` and `@returns`
- Use `process.env.APIFY_API_TOKEN` directly
- Tasks B, C, D create their own `ApifyClient` instance inside the run function
